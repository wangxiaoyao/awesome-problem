## 3 关于 AI

1 AI 会不会取代你的工作的焦虑问题。

- 1 提速的工具
- 2 编写代码的时候占 50%。 其他时间： 懂业务，开会分析需求 => 技术选型：数据库，技术方案。 最重要的是 你作为人，拥有共情能力。而这是 AI 目前不具备的能力。共情能力才能让你利用技术给到用户最佳的体感。
- 3 避免屎山代码。 AI 自动化迭代是增熵的过程。而拥有扎实基本功的程序员需要处理减熵。

2 时代在进步，技术在迭代。 掌握知识，才能不停的走在时代潮流中。

## 6 配置 Vscode

1 调节编辑器: 打开 setting.json。 放入一下

```
    "editor.fontFamily": "Mono Lisa",
    "editor.fontSize": 13,
    "terminal.integrated.fontSize": 13,
    "window.zoomLevel": 1,
    "editor.wordWrap": "on",
    "files.autoSave": "afterDelay"
```

2 安装 extensions

3 shortcuts

- command+p: 文件（快速切换文件）
- command + shift + p: 执行 extensions，等配置和命名。 （等价于上述输入前缀 ">"）
- command + , : 打开设置. 也可以点击右上角一个文件 Open settings（JSON），其配置等价。
- control + ` : 切换内置 terminal。 注意是：英文输入法
- command + b : 切换左侧边栏
- command + +/- : 缩放
- command + w : 关闭文件
- command + x ： 行的任意位置剪切这一行（不仅仅剪切）
- command + 上下左右: 快速的移动光标
- command + shift + 上下左右： 选中整行
- command + d: 相同单词的多个光标选中

- option + 上下左右： 移动代码行

- tab: 可以进行跳转以及补全
